# AquaControl

A management system for swimming instructors built with Java and Spring Boot.

## Features

- Student management
- Attendance tracking
- Group management
- Payment management (planned)
- Attendance statistics (planned)

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Status

- Currently under development.

This project is being developed following an Agile methodology using incremental sprints.
